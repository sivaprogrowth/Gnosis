# raw/assets/

Images, screenshots, and other binary attachments referenced by notes in `raw/articles/` and the wiki.

## How files land here

- **Paste an image** into any note → Obsidian saves it here (via `attachmentFolderPath: raw/assets` in `.obsidian/app.json`).
- **Drop an image** onto a note → same.
- **Localize remote images** in a clipped article → press **Cmd+Shift+D** (Mac) / **Ctrl+Shift+D** (Windows/Linux) to run the Local Images Plus plugin on the current note. It downloads every remote image and rewrites the links to point here.

Same rule as the rest of `raw/`: the LLM never writes here. Only Obsidian + the user.
